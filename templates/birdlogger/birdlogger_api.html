Bird Logger – API docs v1.0





geodata

/api/1.0/geodata[?fromDate=DATE1][&toDate=DATE2]



Response: a GeoJSON data structure or a JSON formatted error message



Parameters:

  • DATE1,DATE2: (type:date) dates in the format YYYY-MM-DD[-h-m-s] (the eventual missing elements are interpreted as zeros).



Possible cases:

  • Both dates are missing: returns the geodata for all the posts in the database

  • fromDate missing: returns data from the beginning to DATE2

  • toDate missing: returns data from DATE1 on.

  • Else: returns data between DATE1 and DATE1



Errors:

  • 400 – Malformed parameter (DATE1 or DATE2): the parameter passed is not of the correct type





post

/api/1.0/post?postId=POST_ID



Response: a GeoJSON data structure or a JSON formatted error message



Parameters:

  • POST_ID: (type:long,required) The ID of the post to retrieve.



Errors:

  • 400 – Missing parameter postId

  • 400 – Malformed parameter postId: the parameter passed is not of the correct type

  • 404 – File not found: the post POST_ID does not exists




image

/api/1.0/image?imageId=IMAGE_ID



Response: an image data or a JSON formatted error message



Parameters:

  • IMAGE_ID: (type:long,required) The ID of the IMAGE to retrieve.



Errors:

  • 400 – Missing parameter imageId

  • 400 – Malformed parameter imageId: the parameter passed is not of the correct type

  • 404 – File not found: the image IMAGE_ID does not exists
